# CodyBird
Old school console/bash clone of famous game Flappy bird written in c++

**Demo:**

[![IMAGE ALT TEXT HERE](http://img.youtube.com/vi/G7Hhvyg_wOY/0.jpg)](http://www.youtube.com/watch?v=G7Hhvyg_wOY)

**Control:**

[SPACE] = Jump

[R] = Reset

[P] = Pause

[Q] = Quit
